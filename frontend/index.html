Consorcio Dashboard

Subí una liquidación y convertí el PDF en un dashboard entendible.

El archivo se envía al parser, se procesa en Lambda y vuelve listo para mostrar administración, consorcio y la distribución de cuánto le corresponde pagar a cada unidad.

Arrastrá el PDF acá o hacé click para subirlo
Se enviará al endpoint real, convertido a base64, y el dashboard se va a renderizar con la respuesta.
